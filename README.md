 # Sign Language Detector

A real-time hand gesture and sign language recognition system developed using Python, OpenCV, and MediaPipe. The application uses webcam input to detect and track hand landmarks, enabling accurate recognition of multiple hand gestures in real time.

## Features
- Real-time hand tracking and gesture detection
- Multi-hand recognition support
- Gesture classification using finger landmark positions
- Detection of gestures such as HELLO, YES, OK, LOVE, FIST, and more
- Lightweight and efficient computer vision implementation

## Technologies Used
- Python
- OpenCV
- MediaPipe
- NumPy

## Applications
- Human-Computer Interaction (HCI)
- Sign Language Interpretation
- AI-based Gesture Control Systems
- Computer Vision Applications
